# Análise de dados da Copa do Mundo (API da FIFA)

Scripts em Python que extraem jogos da API pública da FIFA e fazem análises.

## Scripts
- `fifa_extrair_jogos.py` — baixa os jogos (todas as edições) para JSON e CSV
- `fifa_jogos_de_hoje.py` — palpites e resultados dos jogos do dia
- `fifa_simular_campeao_2026.py` — simula o campeão de 2026 (Monte Carlo)
- `fifa_analise_gols.py` — gráficos de gols e goleadas

## Como usar
1. `pip install requests matplotlib pandas`
2. `python fifa_extrair_jogos.py`
3. Rode os demais scripts conforme a necessidade.

> A API da FIFA não é oficialmente documentada para terceiros e pode mudar sem aviso.# EstudoFIFA
